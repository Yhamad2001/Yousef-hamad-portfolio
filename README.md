### Customer Churn & Revenue Risk Analysis
**Tools:** SQL · Python · Tableau · Excel

End-to-end churn analytics pipeline covering 20 accounts and $46,705 in active MRR.
SQL models risk from behavioral data, Python assigns churn probability scores via
logistic regression, Tableau visualizes the risk landscape, and Excel delivers a
live what-if scenario model.

- [GitHub Repo](https://github.com/Yhamad2001/Customer-Churn-Revenue-Analysis)
- [Live Tableau Dashboard](PASTE_TABLEAU_URL_HERE)

**Key finding:** Reducing churn from 40% to 25% recovers $93,410 in annual revenue.
